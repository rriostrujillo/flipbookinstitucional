FlipBook

Visor de documentos PDF interactivo como Issuu

¿Qué es FlipBook?

FlipBook es una librería que te permite mostrar documentos PDF como revistas o libros interactivos, con efecto de pasando páginas como Issuu. Es embebible en cualquier página web mediante iframe.

📖

Efecto Realista

Animación de pasar páginas suave y natural

📱

Responsive

Funciona en móvil, tablet y desktop

🔗

Embeddable

Incrustable en cualquier sitio web

⚡

Ligero

Solo ~50KB, sin dependencias pesadas

Cómo Usar

Simplemente agrega un iframe apuntando a tu servidor FlipBook con el parámetro pdf indicando la URL del documento: 

<iframe 
  src="https://tu-servidor.com/?pdf=https://ejemplo.com/documento.pdf"
  width="100%" 
  height="500" 
  frameborder="0"
></iframe>

Parámetros Disponibles

Parámetro Descripción Ejemplo
pdf URL del archivo PDF ?pdf=https://...
page Página inicial &page=3
title Título del documento &title=Mi Revista
controls Mostrar controles (1/0) &controls=0
fullscreen Habilitar fullscreen (1/0) &fullscreen=1

Comunicación Padre → Hijo

Puedes controlar el FlipBook desde la página padre usando postMessage: 

// Ir a una página específica
iframe.contentWindow.postMessage({
  type: 'gotoPage',
  data: { page: 5 }
}, '*');

// Obtener estado
iframe.contentWindow.postMessage({
  type: 'getStatus'
}, '*');

// Zoom
iframe.contentWindow.postMessage({
  type: 'zoomIn'
}, '*');
